たべものレスキューとは

九州産業大学の芸術学部×理工学部×地域共創学部の学生が食品ロス削減について，共同で取り組んでいる文理芸融合のプロジェクト型教育です．

もったいない通信とは

このHPでは食品ロスについての情報や私達たべものレスキューが行っている活動について発信していきます．

活動報告

  1. 2021年3月30日に舞松原公民館にて親子で参加によるワークショップを実施しました。ワークショップでは、食品ロスについてのカードを作成し、 そのカードを用いたクイズ大会、また廃棄される食材を使ったジェラートの試食を行いました。

  2. 2022年10月30日にまちスポ福岡東にてワークショップを実施しました。前回と同様に親子参加によるカードとあわせて楽しめるストーリー調のクイズ大会を行いました。

お知らせ

フードドライブを学内に設置しました！！

フードドライブ設置場所：基礎教育センター1号館2階

持ち込み可能な食品
  1. 賞味，消費期限が1ヶ月以上あるもの．
  2. 未開封，未使用なもの．
  3. 生鮮食品や生菓子でないもの．(野菜・コメは除く)
  4. 常温で保存できるもの．
